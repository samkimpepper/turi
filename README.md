# Tech Stack

# Description
투리는 여행 리뷰 SNS 애플리케이션입니다. 사용자가 포스팅한 리뷰들을 지도에서 바로 살펴볼 수 있습니다. 맛집, 숙소, 놀거리, 관광지의 네 가지로 분류하여 검색할 수 있습니다. 
여행 스팟에 특화된 리뷰가 모여있는 SNS입니다.
